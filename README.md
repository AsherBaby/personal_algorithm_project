### Inversion Count of Two Arrays


Inversion count of two arrays implemented with Divide and Conquer
algorithm.

Example:

    A = [1, 3, 2]
    B = [2, 3, 1]
    #inv is 3.

The smaller #inv, the more similar of the two arrays.

This algorithm can be used for recommendation system.
